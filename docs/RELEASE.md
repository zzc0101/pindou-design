## 2. 应用描述（500 字以内）

**功能简介**：
- 📷 一键转换：选图片（含拍照）→ 自动生成拼豆图纸
- 🎨 6 个内置色板：MARD 漫漫 / COCO / 盼盼 / 咪小窝 / Perler / Hama
- 🧊 立体预览：施工前看到接近真实的效果
- ✏️ 手动调整：点击格子改色，撤销/重做自由切换
- 📚 图纸库：本地保存最近 10 张图纸
- 🖨 分页打印：自动分成 A4 多页（含拼接标记）
- 🌙 暗色模式：跟随系统 / 手动切换

**适用人群**：拼豆爱好者、像素画师、手工创作者、家长带孩子玩拼豆。

**核心优势**：
- 算法基于 CIE Lab + CIEDE2000 国际标准色彩匹配，比 RGB 匹配更准
- 同色行扫描合并渲染，100×100 网格 < 1.5s
- 1.78KB 包体积（小程序端，远低于 2MB 主包限制）
- 0 网络依赖，所有算法本地运行

## 9. 更新日志（首次上架 v0.1.0）

### 0.1.0 (2026-06-22)

**功能**：
- 图片转拼豆图纸（6 个色板，289-291 色）
- 实时预览 + 高清图导出
- 手动调整像素 + 撤销/重做
- 网格分块打印（A4）
- 图纸库（本地保存 10 张）
- Floyd-Steinberg 抖动 / 背景移除
- 立体豆子视觉（bead3D）
- KD-Tree 量化（支持 1000+ 色）
- 暗色模式 / 5 个内置模板 / 自定义色板导入

**已知限制**：
- 仅支持 RGBA 图片
- 处理图最大 2048px（自动压缩）
- 不支持云端同步
